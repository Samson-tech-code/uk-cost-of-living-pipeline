# Uk-cost-of-living-pipeline

Automated Python ETL pipeline analysing UK cost of living trends using ONS data, PostgreSQL, and Power BI.



\## Progress

\- \*\*Day 1:\*\* Repository setup, folder structure, configuration, and dependencies.

\- \*\*Day 2:\*\* Automated extraction of ONS time series (CPIH inflation: L55O, wage growth: KAC3) into `data/raw/` with pipeline logging.
\- \*\*Day 3:\*\* Transformed raw ONS time series into a clean long-format dataset (`ons_series_long.csv`) and engineered features (YoY change, 3-month rolling average).
